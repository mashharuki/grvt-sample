# grvt-sample

grvt調査・検証用リポジトリ

## 概要

Grvt（グラビティ）とは、2026年にリリースされる予定のDEX（分散型取引所）で、「ハイブリッド取引所」を目指しています。

ハイブリット取引所とは、DEX（分散型取引所）とCEX（仲介集権型取引所）の2つの特徴を融合した次世代プロジェクトのことです。

## Agent SKILL

```bash
npx skills add gravity-technologies/grvt-skills
```

## APIキーの取得

https://grvt.io/exchange/perpetual/BTC-USDT にアクセスしてサインアップ後発行できるようになる。

以下のページから取得できる

https://grvt.io/exchange/account/api-keys
