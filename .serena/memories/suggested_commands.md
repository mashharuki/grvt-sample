# Suggested Commands

- Install GRVT skills from README:
  - `npx skills add gravity-technologies/grvt-skills`
- Install Python SDK used by local skill examples:
  - `pip install grvt-pysdk`
- Configure environment before account/trading/vault tasks:
  - `export GRVT_ENV="testnet"`
  - `export GRVT_TRADING_API_KEY="..."`
  - `export GRVT_TRADING_PRIVATE_KEY="..."`
  - `export GRVT_FUNDING_API_KEY="..."`
  - `export GRVT_FUNDING_PRIVATE_KEY="..."`
- Useful repo inspection:
  - `rg --files`
  - `git status --short`
- Memory sanity check after onboarding or memory edits:
  - `serena memories check`
- No repo-local build/test/lint command is currently defined.
